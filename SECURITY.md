# Security Policy

## Supported Scope

This repository is currently a prototype provider for Netgear Plus switches and is scoped to `GS108Ev3`.

Security fixes will be evaluated on a best-effort basis for the current `main` branch.

## Reporting A Vulnerability

Please avoid filing public issues for vulnerabilities that could expose credentials, device access, or unsafe switch-management behavior.

If GitHub private vulnerability reporting is enabled for this repository, use that channel. Otherwise, open an issue with only minimal detail and request a private follow-up before sharing reproduction steps or sensitive data.
